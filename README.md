# DV Support App

A mobile application providing support resources for domestic violence survivors, built with React Native and Expo.

## Features

- 24/7 Helpline access with quick dial
- Emergency contacts management
- Personal safety plan creator
- Educational resources
- Quick exit feature for user safety
- All data stored locally on device for privacy

## Installation

```bash
npm install --legacy-peer-deps
```

## Running the App

```bash
npm start
```

Then scan the QR code with Expo Go app on your phone.

## Technologies

- React Native
- Expo SDK 54
- React Navigation
- AsyncStorage
